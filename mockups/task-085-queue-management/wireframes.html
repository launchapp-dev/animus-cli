State machine

Valid queue status transitions:

  • pending → hold → held
  • held → release → pending
  • pending → daemon assigns → assigned
  • assigned → workflow completes → removed from queue

The queue status is independent from task status in .ao/state/tasks.json. A task can be held in the queue while its task record is still ready.
